About the channel

EthanUnlocked's gaming hits

EthanUnlocked's gaming hits is a YouTube content creator and web developer who runs TheUnlockedWeb. He makes gaming videos, shorts, LEGO animations, and tutorials — and has built two web games from scratch: a Cookie Clicker and a Brawl Stars guessing game. This entire website is hand-coded and open source.

YouTube

Gaming videos, speedruns, and shorts covering Brawl Stars, Geometry Dash, and more. New uploads regularly.

Web Games

Two browser games built entirely by Ethan — Cookie Clicker and Brawl Stars-dle, a daily Wordle-style guessing game.

LEGO Animations

Original stop-motion LEGO animations created and edited by Ethan, posted alongside main gaming content.

Community

An active Discord server where fans can chat, share clips, and stay up to date with everything on the channel.

Open Source

All code for this site is public on GitHub. Ethan built and maintains it himself — check it out and see how it works.
