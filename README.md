# TheUnlockedWeb

Personal website for [EthanUnlocked's gaming hits](https://www.youtube.com/@ethan1957-p2y) — hosted on GitHub Pages.

---

## Structure

```
/
├── index.html              # Homepage
├── style.css               # Shared stylesheet (all pages import this)
├── README.md
├── fetch_videos.py         # Script called by the GitHub Action
│
├── .github/
│   └── workflows/
│       └── fetch-videos.yml  # Runs every 6 hours, calls fetch_videos.py
│
├── json/
│   ├── news.json           # News feed data (edit manually)
│   ├── games.json          # Games list (edit manually)
│   └── videos.json         # Auto-generated by GitHub Action
│
├── games/
│   ├── index.html          # Games page
│   └── thumbnails/         # Game thumbnail images
│       └── agar.png
│
└── videos/
    └── index.html          # Videos page
```

All pages use **clean URLs** — no `.html` in links. GitHub Pages serves `videos/index.html` when you visit `/videos/`.

## Pages

### `/` — Homepage
- Animated hero with time-aware greeting (morning / afternoon / evening / night)
- YouTube and Discord CTA buttons with scrolling ticker banners
- About section with cards
- Paginated news feed loaded from `json/news.json`

### `/videos/` — Videos
- Loads from `json/videos.json` (auto-generated by the GitHub Action)
- Click any card to watch in an iframe launcher right on the page
- "Open on YouTube" fallback if the embed is blocked
- Filter by All / Videos / Shorts
- Skeleton loading cards while fetching
- Published time shown as relative (e.g. "3 months ago")

### `/games/` — Games
- Loads from `json/games.json`
- Click any card to play in a fullscreen iframe launcher
- Supports thumbnails from `games/thumbnails/` with icon fallback
- "Open tab" button if the game blocks iframes

---

## Videos — GitHub Action

`json/videos.json` is generated automatically by `.github/workflows/fetch-videos.yml`, which runs `fetch_videos.py` every 6 hours.

The script scrapes your YouTube channel page directly — no API key needed.

**To trigger manually:** GitHub repo → Actions tab → Fetch YouTube Videos → Run workflow.

## Adding News

Open `json/news.json` and prepend a new object to the top of the array:

```json
{
  "id": 13,
  "tag": "YouTube",
  "color": "gold",
  "title": "Your news title here",
  "body": "A short description of the news item.",
  "date": "2025-06-15",
  "icon": "fa-brands fa-youtube"
}
```

| Field | Values |
|-------|--------|
| `tag` | Any short label, e.g. `YouTube`, `Games`, `Site` |
| `color` | `gold`, `purple`, or `green` |
| `date` | ISO format: `YYYY-MM-DD` |
| `icon` | Any [Font Awesome 6](https://fontawesome.com/icons) class string |

## Adding Games

Open `json/games.json` and add an entry:

```json
{
  "id": "mygame",
  "title": "My Game",
  "desc": "Short description of the game.",
  "url": "https://my-game-url.com",
  "thumbnail": "./thumbnails/mygame.png",
  "icon": "fa-solid fa-gamepad",
  "iconBg": "rgba(129,140,248,.13)",
  "iconColor": "#818cf8",
  "tags": [
    { "label": "Puzzle", "color": "purple" }
  ]
}
```

- **Thumbnail:** save an image to `games/thumbnails/mygame.png`. Set `"thumbnail": null` to use the icon fallback instead.
- **Tag colors:** `"gold"`, `"purple"`, or `"green"`.
- **`url`** must allow iframe embedding. If it blocks iframes, the launcher shows an "Open tab" button.

## Shared Styles

`style.css` contains everything shared across pages — tokens, reset, aurora/grain background, nav, footer, buttons, chips, about cards, news paginator, and keyframe animations.

Page-specific styles live in a `<style>` block in each page's `<head>`.

## Adding a New Page

1. Create a directory, e.g. `about/`
2. Add `about/index.html`
3. Link shared assets with `../` paths
4. Copy the nav and footer from an existing page, updating the `.active` link
5. Push — GitHub Pages serves it at `/about/`
